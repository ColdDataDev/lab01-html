# CSS Lab 02

## Опис
Багатосторінковий сайт, оформлений за допомогою CSS.

## Структура проєкту
- index.html
- pages/about.html
- pages/contact.html
- styles/style.css
- styles/responsive.css

## Реалізовано
- Підключення зовнішніх CSS-файлів
- Єдина кольорова схема (змінні :root)
- Оформлення header, footer та навігації
- Flexbox для меню
- Стилізація тексту, списків, таблиці
- Оформлення форми (input, button, focus)
- Оформлення зображень, відео та аудіо
- Адаптивність (media query для 768px)

## Використано
- CSS змінні
- Flexbox
- Media queries

## Запуск
Відкрити index.html у браузері
